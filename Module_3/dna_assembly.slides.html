DNA assembly using De Bruijn graphs¶

Today we will assemble a string of text:

  • without fragmenting it, but only varying its level of complexity
  • fragmenting it in two different ways

def extract_k_mer(sequence,k):
    k_mer = []
    for i in range(len(sequence)-k+1):
        k_mer.append(sequence[i:k+i])
    return k_mer
In [54]:
def de_bruijn_graph(k_mer):
    nodes = set()
    edges = []
    for k in k_mer:
        k1 = k[:-1]
        k2 = k[1:]
        nodes.add(k1)
        nodes.add(k2)
        edges.append((k1,k2))
    return (nodes, edges)

We have an eulerian path within a directed and connected graph, if and only if at most 2 nodes are semi-balanced and all other nodes are balanced.

A node is balanced if indegree equals outdegree.

A node is semi-balanced if indegree differs from outdegree by 1.
